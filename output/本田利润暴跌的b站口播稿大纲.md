# 第三部分：战略失误深度分析 (约3分钟) ⭐⭐⭐

## 3.1 氢能源路线押注失败 ⭐⭐⭐ （来源：王局拍案）

2013年（关键节点）
与 通用汽车
开始合作开发氢燃料电池系统
2017年
成立合资公司（燃料电池量产）

👉 这是全球最早的车企级氢能合作之一

### 3.1.1 技术领先但市场惨败

- 2016年推出氢能源汽车Clarity
- 续航750公里，加氢仅需3分钟
- 价格：755万日元
- **年销量仅200辆**

### 3.1.2 基础设施瓶颈

- 建一个加氢站成本：**1亿日元以上**
- 加氢站网络无法普及
- 消费者不敢买

### 3.1.3 错失转型时机

- 当中国全面拥抱电动车时，本田还在坚持氢能源，与 通用汽车 合作电池
  共同开发下一代电池（cell + module）
  目标：更高能量密度、更低成本

👉 本田将直接采购GM电池系统

📌 关键含义：

👉 本田当时的状态是：

没有自研电池能力
完全依赖外部（GM）

- 而此时2016年中国新能源车开始具备竞争力，举出此时的中国的例子？
- 本田的转型比中国车企慢了5-10年

## 3.2 电动车战略摇摆 ⭐⭐⭐

### 3.2.1 2018年：与通用汽车合作

- 合资开发电池
- 联合研发电动汽车

### 3.2.2 2021年：新CEO三部敏宏上任

- 1987年进入本田，工程师出身
- 敏锐意识到电动车领域已落后
- 提出激进目标：**2040年全部电动化**

### 3.2.3 2022年：雄心勃勃的计划

- 投资**50亿美元**
- 2026年美国推出三款电动车：HONDA 0、HONDA 0 SUV等
- 概念车设计相当漂亮

### 3.2.4 2025年：全部终止 ⭐⭐⭐

- 三款电动车**全部下马**
- 50亿美元投资**打水漂**
- 壮士断腕，全面退出

## 3.3 美国政策突变（来源：王局拍案 ⭐⭐⭐）

### 3.3.1 加州清洁能源法案失效

- 原法案：2025年50%零排放，2030年60%，2035年100%
- 特朗普上台后：法案被宣告无效

### 3.3.2 电动车补贴退坡

- 2024年9月：美国电动车补贴**全面取消**

### 3.3.3 销量预期落差

- 本田预计：美国市场年销量差距**150万辆**
- EV需求不及预期（核心原因）
  美国EV增速放缓
  消费者更偏向：
  混动
  燃油车

👉 本田判断：

👉 “卖不动”

  三、丰田有没有认真研究电动车？

答案是：

👉 不仅研究了，而且研究得很深，但商业化节奏很慢

👉 但都没规模化

📌 原因：
👉 内部并不看好短期EV

🔹 3️⃣ 真正加速：2021年以后

与本田类似，2021年前后：

👉 丰田开始明显加码EV：

发布 bZ系列（bZ4X）
投资数万亿日元电池研发
建电池工厂（全球布局）
🔹 4️⃣ 押注“终极技术”：固态电池

这是丰田最重要的一张牌：

👉 固态电池

优势：

更高能量密度
更快充电
更安全

👉 丰田在这个领域专利全球领先

📌 战略意图：

👉 不在当前锂电池卷
👉 直接跳下一代
