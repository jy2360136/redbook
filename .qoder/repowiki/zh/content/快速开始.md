## 简介

Redbook是一个专业的新闻抓取和社区分析工具，专注于为中国科技和财经领域提供高质量的内容采集和分析服务。该项目集成了多种先进的爬虫技术和AI驱动的分析能力，能够帮助用户快速获取市场动态、分析社区舆情，并生成有价值的内容洞察。

项目的核心功能包括：
- **专业新闻抓取**：从7大权威财经媒体自动抓取热点新闻
- **社区论坛分析**：收集雪球、知乎等平台的用户评论和讨论
- **AI增强抓取**：利用Crawl4AI技术处理复杂的网页结构
- **情感分析**：自动分析用户评论的情感倾向
- **多格式输出**：支持JSON等多种输出格式

## 专业新闻抓取

### 基本语法

```bash
python financial_news_workflow_crawl4ai.py --days <天数> --sources <来源>
```

### 常用示例

#### 抓取近10天的虎嗅网和第一财经新闻
```bash
python financial_news_workflow_crawl4ai.py --days 10 --sources huxiu,yicai
```

#### 抓取所有来源的新闻
```bash
python financial_news_workflow_crawl4ai.py --days 7 --sources all
```

#### 指定输出目录
```bash
python financial_news_workflow_crawl4ai.py --days 5 --sources huxiu,36kr --output ./output
```

#### 使用固定输出目录
```bash
python financial_news_workflow_crawl4ai.py --days 3 --sources yicai,jiemian --fixed-output ./news_output
```

### 参数说明

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `--days` | 整数 | 3 | 抓取近X天的新闻 |
| `--sources` | 字符串 | "all" | 新闻来源，用逗号分隔 |
| `--output` | 字符串 | "." | 输出基础目录 |
| `--filter-companies` | 布尔值 | False | 是否启用公司名过滤 |

### 支持的新闻源

项目支持以下7大权威财经媒体：

1. **虎嗅网** (RSS) - 科技热点新闻
2. **晚点 LatePost** (Playwright) - 深度科技报道
3. **36氪** (API) - 创业投资资讯
4. **钛媒体** (RSS) - 科技产业观察
5. **极客公园** (Playwright) - 科技前沿动态
6. **界面新闻** (RSS) - 财经时事报道
7. **澎湃新闻** (requests) - 综合新闻资讯

### 输出结果

运行完成后，系统会在当前目录创建一个带时间戳的输出目录，包含以下文件：

- `news_result.json`：包含抓取的新闻数据
- `prompt.txt`：生成的分析提示词

## 社区论坛抓取

### 基本语法

```bash
python community_crawler.py --keyword <关键词> --sources <来源>
```

### 常用示例

#### 抓取关于"小米汽车"的所有社区评论
```bash
python community_crawler.py --keyword "小米汽车" --sources all
```

#### 只抓取雪球的评论
```bash
python community_crawler.py --keyword "华为" --sources xueqiu
```

#### 指定输出目录
```bash
python community_crawler.py --keyword "腾讯" --sources zhihu --output ./community_output
```

### 参数说明

| 参数 | 类型 | 默认值 | 描述 |
|------|------|--------|------|
| `--keyword` | 字符串 | 必填 | 搜索关键词 |
| `--sources` | 字符串 | "all" | 社区来源，用逗号分隔 |
| `--output` | 字符串 | "." | 输出基础目录 |

### 支持的社区平台

项目支持以下社区论坛：

1. **雪球网** (xueqiu.com) - 专业投资社区
2. **知乎** (zhihu.com) - 综合问答社区

### 输出结果

运行完成后，系统会在输出目录创建一个带时间戳的子目录，包含以下文件：

- `comments_<关键词>.json`：包含抓取的评论数据和情感分析结果

### 情感分析功能

系统内置简单的情感分析功能，基于关键词匹配进行情感判断：

- **积极情感**：包含积极词汇（如"好"、"棒"、"优秀"等）
- **消极情感**：包含消极词汇（如"差"、"糟糕"、"失望"等）
- **中性情感**：积极和消极词汇数量相等

## 常见问题解答

### Q1：为什么有些新闻源抓取不到数据？

**A1**：这可能有以下几个原因：
- 网站结构发生变化
- 网站设置了反爬虫机制
- 当前时间段没有相关数据
- 网络连接不稳定

**解决方法**：
```bash
# 尝试使用不同的时间范围
python financial_news_workflow_crawl4ai.py --days 1

# 指定特定的新闻源
python financial_news_workflow_crawl4ai.py --sources huxiu
```

### Q2：如何提高抓取速度？

**A2**：可以通过以下方式优化：
- 减少抓取的天数范围
- 限制使用的新闻源数量
- 适当增加请求间隔
- 使用更强大的硬件配置

### Q3：如何处理验证码？

**A3**：项目已经集成了多种反爬虫应对策略：
- 使用随机User-Agent
- 实现智能等待机制
- 支持Crawl4AI的AI增强抓取
- 自动处理JavaScript渲染

### Q4：如何自定义输出格式？

**A4**：目前项目支持JSON格式输出。如果需要其他格式，可以：
- 修改保存函数
- 添加新的输出处理器
- 扩展数据处理逻辑

### Q5：如何添加新的新闻源？

**A5**：添加新新闻源的步骤：
1. 在相应的类中实现抓取逻辑
2. 更新源映射表
3. 添加参数解析
4. 测试新功能

## 性能优化建议

   # 实现本地缓存
   import pickle
   
   cache_file = "news_cache.pkl"
   if os.path.exists(cache_file):
