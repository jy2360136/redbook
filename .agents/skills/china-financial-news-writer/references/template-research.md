# 研报简报模板库 (Research Brief Templates)

## 通用规则

### 内容特征
- **字数**: 3000-5000字
- **字体**: Times New Roman（中文用宋体）
- **语气**: 专业、客观、数据密集
- **表格**: 3-5个
- **图表**: 8-12个

### 标题规范

```
标准格式：[公司名]([代码])：[事件]+[核心观点]

示例：
- 比亚迪(002594)：Q3营收超预期，盈利能力持续提升 - 买入
- 小鹏汽车(XPEV)：MONA M03发布，下沉市场战略落地 - 增持
- 腾讯控股(0700.HK)：游戏业务回暖，广告收入超预期 - 买入
```

### 评级体系

| 评级 | 含义 | 预期收益 |
|------|------|----------|
| 买入 | Strong Buy | >20% |
| 增持 | Outperform | 10-20% |
| 中性 | Neutral/hold | 0-10% |
| 减持 | Underperform | -10%-0% |
| 卖出 | Sell | <-10% |

## 图表规范

### 图表编号与标题

```
图表编号格式：图X / 表X
图表标题位置：图表下方
数据来源：图表下方右对齐

示例：
图1: 季度收入趋势（百万元）
Source: 公司财报，XX证券
```

### 常用图表类型

| 分析内容 | 推荐图表 |
|----------|----------|
| 季度趋势 | 柱状图+折线图组合 |
| 年度对比 | 分组柱状图 |
| 结构占比 | 堆叠柱状图/饼图 |
| 同业对比 | 横向柱状图/表格 |
| 估值区间 | 面积图+折线图 |
| 预测数据 | 虚线+实线组合图 |

### 表格规范

```
1. 表头加粗
2. 数字右对齐
3. 文字左对齐
4. 单位统一
5. 数据来源标注
6. 适当使用合并单元格
```
