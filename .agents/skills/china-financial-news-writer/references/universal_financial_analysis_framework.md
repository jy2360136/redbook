# 万能财经分析框架 - 企业危机事件深度分析模板

## 🎯 核心模块（必选）

### 1. 事件引爆点
- [ ] 核心数据冲击（利润暴跌/营收下滑/亏损扩大等具体百分比）
- [ ] 历史性转折点（打破多少年记录/首次出现某种情况）
- [ ] 财务影响规模（具体损失金额/计提损失等）
- [ ] 时间节点重要性（财报季/政策变化/突发事件等）

### 2. 战略失误分析
- [ ] 战略路线错误（技术路线选择/市场定位偏差）
- [ ] 决策时机误判（过早/过晚进入某个领域）
- [ ] 外部环境误判（政策变化/市场需求变化）
- [ ] 竞争对手低估（对新兴竞争者重视不足）
- [ ] 资源投入失误（过度投资/投资不足的具体项目）

### 3. 市场竞争格局
- [ ] 同行业竞争对手对比（直接竞品表现对比）
- [ ] 跨行业替代威胁（新技术/新模式的替代风险）
- [ ] 市场份额变化（具体数据支撑的市场地位变化）
- [ ] 产品竞争力分析（具体产品/服务的优劣势对比）
- [ ] 区域市场表现（不同地区市场的差异化表现）

## 📊 数据分析模块（可选）

### 4. 财务深度分析
- [ ] 利润变化趋势（多季度/多年度对比）
- [ ] 营收结构分析（各业务线贡献度变化）
- [ ] 成本费用分析（成本上升/费用控制问题）
- [ ] 资产负债表健康度（负债率/现金流等关键指标）
- [ ] 投资回报分析（ROE/ROA等核心指标变化）

### 5. 全网舆情分析
- [ ] 媒体态度分布（正面/中性/负面比例）
- [ ] 分析师观点汇总（看多/看空主流观点）
- [ ] 社交媒体情绪（微博/知乎/雪球等平台情绪）
- [ ] 热门评论精选（代表性网友观点摘录）
- [ ] 搜索热度变化（百度指数/微信指数等数据）

## 🔍 深度洞察模块（可选）

### 6. 技术路线分析
- [ ] 核心技术竞争力（专利/研发投入/技术壁垒）
- [ ] 技术迭代速度（与行业平均水平的对比）
- [ ] 技术路线选择（不同技术路径的优劣势）
- [ ] 技术人才储备（核心团队/人才流失情况）
- [ ] 技术生态布局（产业链上下游整合能力）

### 7. 历史对比分析
- [ ] 行业历史周期（类似历史时期的对比分析）
- [ ] 同类企业案例（其他企业类似困境及结果）
- [ ] 国家产业变迁（整个行业/国家层面的历史规律）
- [ ] 技术革命影响（历次技术革命对行业的冲击）
- [ ] 经济周期规律（宏观经济周期对行业的影响）

### 8. 未来预测模块
- [ ] 短期影响预测（1年内可能的发展趋势）
- [ ] 中期趋势分析（1-3年行业格局变化）
- [ ] 长期展望评估（3-5年战略发展方向）
- [ ] 情景分析模型（乐观/中性/悲观三种情景）
- [ ] 关键触发因素（可能改变趋势的关键变量）

## 🎬 内容表达模块

### 9. 故事化叙事
- [ ] 企业发展史回顾（从崛起到危机的关键节点）
- [ ] 关键人物决策分析（CEO/核心管理层的重要决策）
- [ ] 标志性事件时间线（重大事件的时间脉络）
- [ ] 企业文化影响（组织文化对危机的影响）
- [ ] 行业地位变迁（从领导者到追赶者的角色转变）

### 10. 情感共鸣点
- [ ] 代际观念变化（不同年龄层的认知差异）
- [ ] 消费者心理变化（用户需求和偏好的演变）
- [ ] 民族情感因素（国产品牌崛起vs外资品牌衰落）
- [ ] 就业影响关注（对员工/供应商等利益相关方影响）
- [ ] 社会价值讨论（企业社会责任和价值观变迁）

### 11. 互动设计
- [ ] 观众投票问题（设置争议性话题引导投票）
- [ ] 评论区话题引导（预设讨论方向和热点话题）
- [ ] 下期内容预告（关联性话题的延续性设计）
- [ ] 知识问答环节（设置简单问题增加参与感）
- [ ] 案例征集活动（邀请观众分享类似经历）

## 📈 可视化建议

### 12. 图表模板库
- [ ] 财务数据图表（折线图/柱状图/饼图等）
- [ ] 市场份额可视化（饼图/雷达图/热力图）
- [ ] 时间趋势线图（多指标对比的时间序列图）
- [ ] 对比分析表格（SWOT/PEST等分析框架表格）
- [ ] 地理信息图（区域市场表现的地图可视化）

## 🎯 平台适配指南

### 13. 不同平台内容侧重
- [ ] **B站风格**：故事化+数据可视化+互动设计
- [ ] **小红书风格**：个人体验+实用建议+情感共鸣
- [ ] **公众号风格**：深度分析+专业观点+长文阅读
- [ ] **抖音风格**：短视频脚本+金句提炼+视觉冲击
- [ ] **微博风格**：热点话题+争议观点+传播性文案

## 📋 使用指南

### 步骤1：内容勾选
根据具体企业危机事件，勾选相关分析模块

### 步骤2：优先级排序
按目标平台特性，确定内容权重和时长分配

### 步骤3：数据收集
针对选中模块，收集相应的数据和案例支撑

### 步骤4：内容生成
使用AI工具基于大纲生成具体分析内容

### 步骤5：平台适配
根据不同平台调性调整表达方式和内容深度
