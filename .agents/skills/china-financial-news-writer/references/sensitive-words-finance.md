# 金融内容敏感词库 (Sensitive Words for Financial Content)

## 🔴 高危词汇 (封号/处罚风险)

### 投资建议违规

| 违规词 | 问题 | 后果 |
|--------|------|------|
| 必涨、一定涨、肯定涨 | 收益承诺 | 封号/处罚 |
| 稳赚、保本、包赚 | 收益承诺 | 封号/处罚 |
| 内幕消息、内部消息 | 涉嫌内幕交易 | 封号/处罚 |
| 100%盈利、零风险 | 虚假宣传 | 封号/处罚 |
| 荐股、代客理财 | 非法证券业务 | 封号/处罚 |
| 跟单、带单、跟投 | 非法证券业务 | 封号/处罚 |

### 非法集资相关

| 违规词 | 问题 | 后果 |
|--------|------|------|
| 保本保息 | 非法集资特征 | 封号/处罚 |
| 高收益理财 | 非法集资特征 | 封号/处罚 |
| 私募募集 | 非法集资 | 封号/处罚 |
| 年化收益XX% | 收益承诺 | 限流/处罚 |

### 虚假宣传

| 违规词 | 问题 | 后果 |
|--------|------|------|
| 内部渠道、特殊通道 | 虚假宣传 | 限流/处罚 |
| 包过、包赚 | 虚假承诺 | 限流/处罚 |
| 独家内幕 | 虚假宣传 | 限流/处罚 |

## 🟠 中危词汇 (限流风险)

### 绝对化用语

| 违规词 | 替代方案 |
|--------|----------|
| 最好的股票 | 优质股票、值得关注 |
| 最佳投资 | 具有投资价值 |
| 必涨股 | 具有上涨潜力 |
| 100%、绝对 | 较大概率、较高可能 |
| 肯定赚钱 | 值得关注 |
| 稳赚不赔 | 风险收益比较好 |

### 投资建议类

| 违规词 | 替代方案 |
|--------|----------|
| 强烈推荐买入 | 看好、值得关注 |
| 建议买入 | 可以关注 |
| 必须持有 | 具有持有价值 |
| 赶紧买 | 值得考虑 |
| 满仓、梭哈 | 可以配置 |
| 抄底机会 | 估值具有吸引力 |

### 预测类

| 违规词 | 替代方案 |
|--------|----------|
| 必定涨到XX | 可能达到XX |
| 肯定突破 | 有望突破 |
| 一定会 | 较大概率 |
| 保证收益 | 预期收益 |

## 🟡 低危词汇 (可能延迟审核)

### 可能触发审核

| 词汇类型 | 示例 | 注意事项 |
|----------|------|----------|
| 借贷相关 | 贷款、借贷、配资 | 需谨慎表述 |
| 理财产品 | 理财、基金、信托 | 需风险提示 |
| 期货外汇 | 期货、外汇、杠杆 | 需风险提示 |
| 数字货币 | 比特币、虚拟币 | 需风险提示 |
| 募资相关 | 募资、融资 | 需合规表述 |

### 行业特殊词

| 行业 | 敏感词 | 注意事项 |
|------|--------|----------|
| 医药 | 治疗、治愈、疗效 | 避免医疗效果承诺 |
| 白酒 | 功效、保健 | 避免夸大功效 |
| 教育 | 包过、保过 | 避免效果承诺 |

## 平台特殊规则

### 小红书金融内容

```
⚠️ 特别注意：
- 不允许发布荐股内容
- 不允许承诺收益
- 不允许引导开户
- 理财内容需标注风险提示

✅ 允许内容：
- 财报解读（数据+分析）
- 行业分析（客观分析）
- 投资知识科普
- 理财经验分享（个人经验）

❌ 禁止内容：
- 具体股票推荐
- 买入卖出建议
- 收益承诺
- 开户引导
```

### 公众号金融内容

```
⚠️ 合规要求：
- 需标注"不构成投资建议"
- 分析师需有资质
- 禁止承诺收益
- 需披露利益关系

✅ 必须包含：
- 免责声明
- 风险提示
- 数据来源标注
```

### 研报合规

```
⚠️ 严格合规：
- 需分析师署名
- 需投资评级定义
- 需免责声明
- 需披露利益关系

✅ 必须包含：
- 分析师声明
- 投资评级说明
- 完整免责声明
- 公司声明
```

## 风险提示模板

### 小红书风险提示

```
简短版：
"⚠️ 仅供参考，不构成投资建议"

完整版：
"本文仅为个人学习笔记，不构成任何投资建议。
投资有风险，入市需谨慎！"
```

### 公众号风险提示

```
标准版：
"风险提示：本文仅代表作者个人观点，不构成任何投资建议。
投资者应当自行判断投资决策，并承担投资风险。过去业绩不代表未来表现。"
```

### 研报风险提示

```
完整版：
"风险提示：
1. 宏观经济下行风险
2. 行业竞争加剧风险
3. 政策监管风险
4. [公司特有风险]

免责声明：
本报告仅供客户使用，不构成任何投资建议。本报告基于已公开信息编制，
但不保证信息的准确性和完整性。投资者应当自行判断投资决策，并承担
投资风险。过去业绩不代表未来表现。"
```

## 检测方法

### 分析流程

```
1. 全文扫描
   ├── 检查高危词库
   ├── 检查中危词库
   └── 检查低危词库

2. 上下文分析
   ├── 判断使用语境
   ├── 判断是否有违规意图
   └── 判断是否有风险提示

3. 替换建议
   ├── 提供安全替换词
   ├── 提供改写建议
   └── 检查替换后是否通顺

4. 合规确认
   ├── 添加风险提示
   ├── 添加免责声明
   └── 确认符合平台规则
```

### 输出格式

```markdown
## 合规检查报告

**风险等级**: 🟢/🟡/🟠/🔴

**检测到的敏感词**:
| 词汇 | 风险等级 | 出现位置 | 替换建议 |
|------|----------|----------|----------|
| [词汇] | [等级] | [位置] | [替换] |

**修改建议**:
1. [建议]
2. [建议]

**需要添加的风险提示**:
[风险提示内容]

**修改后风险等级**: 🟢
```
