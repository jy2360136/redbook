# 小红书模板库 (Xiaohongshu Templates)

## 通用规则

### 内容特征
- **字数**: 500-800字
- **段落**: 短句为主，每段2-3行
- **emoji**: 适度使用，每50字1-2个
- **语气**: 亲切、口语化、真实感
- **标签**: 5-10个

### 标题公式（金融版）

| 类型 | 公式 | 示例 |
|------|------|------|
| **数字型** | 数字 + 核心结论 | "比亚迪Q3卖爆了！营收破2000亿🔥" |
| **情绪型** | 情绪词 + 关键信息 | "救命！小鹏这波真的要起飞了？" |
| **对比型** | Before/After | "从亏损到盈利，理想做对了什么" |
| **好奇型** | 悬念 + 关键词 | "腾讯财报藏着一个大秘密..." |
| **身份型** | 身份 + 干货 | "券商分析师：财报要看这5个指标" |
| **场景型** | 场景 + 解决方案 | "财报季来了！小白也能看懂的解读" |
| **盘点型** | 合集/清单 | "2024新能源车企销量排行榜📊" |
| **争议型** | 轻度争议观点 | "比亚迪销量第一，但股价为什么不涨？" |

### 首段黄金50字（金融版）

```
公式：场景/痛点(15字) + 核心信息预告(20字) + 继续阅读钩子(15字)

示例1（财报类）:
"财报季又到了！看不懂财报的姐妹看过来，
今天用大白话解读比亚迪Q3财报，
这3个数据最重要👇"

配置居然比20万的还顶？
来算算这车到底值不值🧮"

示例3（行业类）:

往下看分析👇"
```

## 互动设计技巧

### 评论引导

```
1. 提问式
   "你们觉得这个价格贵不贵？"
   "大家会选择哪家？"

2. 投票式
   "评论区投票：🅰️看好 🅱️观望 🅾️不看好"

3. 求助式
   "有懂行的朋友帮忙分析一下吗？"

4. 共鸣式
   "有和我一样一直在观望的吗？"

5. 争议式
   "你们觉得[观点A]还是[观点B]？"
```

### 收藏价值设计

```
高收藏内容特征：
✅ 数据清单（关键指标汇总）
✅ 对比表格（竞品对比）
✅ 时间线（重要节点梳理）
✅ 公式/方法论（分析框架）
✅ 避坑指南（风险提示）
```

### 分享动机触发

```
用户分享的原因：
1. 觉得有用 → 干货型内容
2. 觉得有料 → 独家解读
3. 表达态度 → 观点型内容
4. 社交货币 → "我也懂财经"
```

## 格式规范

### Emoji使用指南

```
适度使用的场景：
📊 数据展示
💰 金钱/投资相关
📈 上涨/利好
📉 下跌/利空
⚠️ 风险提示
🔥 热门/重要
✅ 正面信息
❌ 负面信息
💡 观点/建议
🔍 分析/解读
🎯 核心要点

避免：
❌ 每句话都加emoji
❌ 连续使用3个以上emoji
❌ 与内容不符的emoji
```

### 段落节奏

```
理想节奏：
- 标题：15-20字
- 首段：50字以内
- 要点：每条1-2行
- 段落之间空行
- 总段落：8-12段
```

### 标签布局

```
组合公式：
2-3个核心词标签：#比亚迪 #新能源汽车
2-3个长尾词标签：#比亚迪财报 #新能源车销量
1-2个场景词标签：#投资理财 #股票入门
1-2个人群词标签：#理财小白 #投资笔记
1-2个热门话题标签：#我的理财日记 #好文分享
```
