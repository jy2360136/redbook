# 深度分析报告模板 (Deep Research Report Template)

## 文档说明

本模板用于输出**全网深度调研**后的综合分析报告，适用于重大财经事件的深度解读。

**字数**：3000-8000 字（根据事件复杂度）
**格式**：Markdown
**时效**：事件发生后 24-72 小时内

---

## 报告结构

```markdown
# {公司名}{事件}深度分析报告

**报告日期**: YYYY-MM-DD
**调研时间范围**: {事件发生日} - {报告撰写日}
**信息来源**: {数量}+ 篇媒体报道、{数量}+ 篇专业分析、{数量}+ 条社交媒体讨论

---
```

## 使用指南

### 适用场景

| 场景 | 推荐章节 | 字数 |
|------|----------|------|
| 财报暴跌 | 全章节 | 5000-8000 |
| 产品发布 | 一、三、五、七 | 3000-5000 |
| 政策影响 | 一、二、三、五、七 | 4000-6000 |
| 行业事件 | 一、三、四、五、七 | 4000-6000 |

### 撰写流程

1. **信息收集**（30-60 分钟）：按 deep-research.md 执行
2. **信息整理**（30 分钟）：填入时间线、数据表格
3. **分析撰写**（60-90 分钟）：按模板填充内容
4. **审核校对**（15 分钟）：检查数据、逻辑、合规

### 质量检查清单

- [ ] 所有关键数据有来源标注
- [ ] 时间线清晰准确
- [ ] 多空观点都有呈现
- [ ] 风险提示完整
- [ ] 免责声明包含
- [ ] 无荐股表述
- [ ] 逻辑链条完整

## 示例：本田利润暴跌报告框架

```markdown
# 本田中国利润暴跌 90% 深度分析报告

**核心结论**：
1. 利润暴跌主因：中国销量下滑 35%+ 价格战压缩毛利
2. 深层原因：电动化转型滞后，被比亚迪/特斯拉挤压
3. 短期难反转，需观察 {当前年份+1} 年新车型表现

**时间线**：
- YYYY-MM-DD: {公司名}发布{财报/公告}
- YYYY-MM-DD: 媒体集中报道
- YYYY-MM-DD: {公司名}官方回应

**原因分析**：
- 直接：销量下滑、毛利率下降
- 深层：电动化慢、中国市场竞争加剧
- 外部：日系车整体遇冷、政策向新能源倾斜

**全网观点**：
- 分析师：普遍下调评级
- 社交媒体：悲观情绪主导
- 视频平台："日系车衰落"成热门话题

**风险提示**：
- 中国销量继续下滑
- 电动化转型不及预期
- 汇率波动风险
```
