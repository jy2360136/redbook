## 使用说明

### 如何使用公司画像

```
1. 确认目标公司
   - 查找公司基本信息
   - 确认业务结构

2. 确定关注指标
   - 根据公司类型选择核心指标
   - 补充行业特殊指标

3. 查找数据来源
   - 官方公告
   - 第三方数据

4. 确定写作侧重
   - 结合当前市场热点
   - 突出公司特色
```

### 扩展公司画像

如需添加新公司，按以下模板：

```yaml
基本信息:
  公司全称: [全称]
  股票代码: [代码]
  上市地点: [交易所]
  主营业务: [业务描述]

业务结构:
  - [业务1]: [占比]
  - [业务2]: [占比]

核心关注指标:
  - [指标1]
  - [指标2]

财报发布:
  - 年报: [月份]
  - 一季报: [月份]
  - 中报: [月份]
  - 三季报: [月份]

写作侧重:
  - [侧重1]
  - [侧重2]
```
