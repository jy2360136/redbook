# 公众号模板库 (WeChat Official Account Templates)

## 通用规则

### 内容特征
- **字数**: 1500-2500字
- **段落**: 中等长度，每段4-6行
- **语气**: 专业但不失活泼
- **数据**: 适度使用，配合解读
- **配图**: 5-8张

### 标题公式（公众号版）

| 类型 | 公式 | 示例 |
|------|------|------|
| **事件型** | [公司]+[事件]+核心结论 | "比亚迪Q3财报：营收破2000亿，这个信号最重要" |
| **分析型** | [事件]+深度解读 | "小鹏MONA发布背后：15万级市场的价格战才刚开始" |
| **观点型** | 观点+理由预告 | "为什么我看好比亚迪：三个被忽视的细节" |
| **提问型** | 问题+文章价值 | "新能源车企谁能笑到最后？从Q3财报找答案" |
| **对比型** | A vs B+结论 | "蔚来vs小鹏vs理想：谁才是新势力一哥？" |
| **时效型** | 突发/重磅+事件 | "重磅！央行最新政策，对股市影响几何？" |

### 导语写法（200字）

```
公式：新闻概述(100字) + 核心观点(50字) + 阅读指引(50字)

示例：
"{月}月{日}日，比亚迪发布{年份}年{季度}财报，营收达到2011亿元，
同比增长24%，再创单季新高。但比营收更重要的是，
这份财报透露出三个关键信号，可能改变市场对新能源车竞争格局的判断。
本文将从财报亮点、隐忧、行业对比三个维度深度解读。"
```

## 写作技巧

### 数据呈现

```
1. 大数字要换算
   ❌ "营收201112330000元"
   ✅ "营收2011亿元"

2. 对比要有参照
   ❌ "增长了24%"
   ✅ "同比增长24%，超过行业平均的15%"

3. 用表格整理复杂数据
   [使用表格展示对比数据]

4. 用图表展示趋势
   [使用折线图/柱状图展示趋势]
```

### 观点表达

```
1. 观点要明确
   ❌ "可能会有一定影响"
   ✅ "预计将带来10-15%的增量"

2. 要有数据支撑
   ❌ "这个业务发展很好"
   ✅ "该业务营收增长35%，占总收入比重提升至28%"

3. 承认不确定性
   ❌ "肯定会涨"
   ✅ "若能达成指引，预计将..."
```

### 段落节奏

```
1. 每段一个主题
2. 段落开头用主题句
3. 段落之间用过渡句
4. 适当使用小标题分割
5. 避免大段文字（超过6行）
```

### 互动引导

```
文末互动设计：
1. 提问引导评论
   "你怎么看待这个变化？欢迎评论区讨论"

2. 引导关注
   "关注我们，持续追踪后续发展"

3. 引导转发
   "觉得有用？点个'在看'分享给更多人"
```

## 配图指南

### 图表类型

| 内容类型 | 推荐图表 |
|----------|----------|
| 财务数据对比 | 柱状图/表格 |
| 趋势变化 | 折线图 |
| 占比结构 | 饼图/堆叠图 |
| 竞品对比 | 对比表格 |
| 产业链 | 流程图 |
| 时间线 | 时间轴图 |

### 配图数量

```
- 导语配图: 1张（新闻图/公司图）
- 数据图表: 3-5张
- 概念图/示意图: 1-2张
- 总计: 5-8张
```

### 图片规范

```
1. 图片清晰度：宽度至少1080px
2. 图片比例：16:9 或 4:3
3. 配文字说明
4. 标注数据来源
```
