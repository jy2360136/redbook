# 金融内容关键词策略 (Keyword Strategy for Financial Content)

## 关键词类型

### 1. 核心关键词 (Core Keywords)
用户搜索的主要词汇，竞争激烈但流量大

**特征**:
- 1-4个字/词
- 高搜索量
- 竞争激烈

**金融领域示例**:
- 公司名：比亚迪、腾讯、茅台、宁德时代
- 行业名：新能源、白酒、半导体、医药
- 概念名：财报、估值、财报季、投资

### 2. 长尾关键词 (Long-tail Keywords)
更具体的搜索词，竞争小但精准

**特征**:
- 5-15个字
- 搜索量较小
- 用户意图明确
- 转化率高

**示例**:
- 核心词"比亚迪" → 长尾词"比亚迪Q3财报解读"、"比亚迪销量分析"
- 核心词"新能源" → 长尾词"新能源车投资逻辑"、"新能源车企估值对比"

### 3. 场景关键词 (Scenario Keywords)
描述使用场景的词汇

**金融场景示例**:
- 财报季、年报披露、业绩发布
- 新股上市、定增、回购
- 降准、降息、政策发布
- 除权除息、年报行情

### 4. 人群关键词 (Audience Keywords)
描述目标人群的词汇

**金融受众示例**:
- 股民、投资者、散户
- 理财小白、新手入门
- 价值投资者、短线客
- 基民、定投党

### 5. 修饰关键词 (Modifier Keywords)
增加描述的词汇

**金融修饰词示例**:
- 情绪类：重磅、突发、终于、意外
- 程度类：深度、全面、详解、干货
- 时间类：最新、年份、Q3、实时（如：{年份}年Q3）
- 价值类：必看、收藏、避坑、实战

## 关键词布局策略

### 布局位置优先级

```
1. 标题 (权重最高) - 必含核心词
2. 首段前100字 (权重高) - 核心词+1-2个长尾词
3. 小标题 (权重中) - 长尾词/场景词
4. 正文内容 (权重中) - 自然分布
5. 标签/话题 (权重中) - 混合布局
6. 图片OCR文字 (权重低) - 补充
```

### 最佳布局公式

#### 小红书布局

```
标题: [情绪词] + [公司/行业核心词] + [数字/结论] + [emoji]
      例: "比亚迪Q3财报炸了！营收破2000亿🔥"

首段(50字内): 核心关键词 + 痛点/场景 + 继续阅读钩子
      例: "比亚迪Q3财报出炉，营收直接破2000亿！
          这三个数据最重要，小白也能看懂👇"

正文: 每150-200字自然出现1次关键词变体
      例: "比亚迪的销量"、"新能源车龙头"

标签: 5-10个，覆盖不同类型
      例: #比亚迪 #新能源车 #财报解读 #投资理财 #股票入门
```

#### 公众号布局

```
标题: [公司/行业核心词] + [事件/变化] + [价值点]
      例: "比亚迪Q3营收破2000亿：这三个信号最重要"

导语(200字内): 核心词 + 事件概述 + 文章结构预告
      例: "{月}月{日}日，比亚迪发布{年份}年{季度}财报，
          营收达到2011亿元，同比增长24%。
          本文将从财报亮点、隐忧、行业对比三个维度解读。"

小标题: 包含长尾关键词
      例: "01 核心数据速览"、"02 业务结构拆解"

正文: 每段自然出现相关关键词
      例: "比亚迪新能源车销量"、"新能源汽车市场"

标签/话题: 3-5个
      例: #比亚迪 #财报分析 #新能源汽车
```

#### 研报布局

```
标题: [公司全称]([代码]) + [事件] + [观点] - [评级]
      例: "比亚迪(002594)：Q3营收超预期，盈利能力提升 - 买入"

投资要点: 核心词 + 核心结论
      例: "比亚迪Q3营收2011亿元，超市场预期3%"

小标题: 专业术语 + 长尾词
      例: "2.1 收入分析"、"2.2 盈利能力分析"
```

## 标签策略

### 小红书标签组合

```
2-3个 核心词标签: #比亚迪 #新能源车
2-3个 长尾词标签: #比亚迪财报 #新能源车销量
1-2个 场景词标签: #财报季 #投资理财
1-2个 人群词标签: #股票入门 #理财小白
1-2个 热门话题标签: #好文分享 #我的理财日记

总计: 5-10个
```

### 公众号话题组合

```
1-2个 核心词话题: #比亚迪 #新能源汽车
1-2个 长尾词话题: #财报分析 #投资研究

总计: 3-5个
```

### 标签注意事项

✅ **正确做法**:
- 标签与内容高度相关
- 混合大词和长尾词
- 使用平台热门话题标签
- 包含品牌/公司标签

❌ **错误做法**:
- 堆砌不相关热门标签
- 只用大词不用长尾词
- 标签与内容不符
- 过多标签（>15个）

## 关键词密度

### 理想密度

| 平台 | 核心关键词密度 | 长尾关键词 |
|------|----------------|------------|
| 小红书 | 2-3% (每100字2-3次) | 每个出现1-2次 |
| 公众号 | 1.5-2.5% | 每个出现2-3次 |
| 研报 | 1-2% | 每个出现3-5次 |

### 检查方法

```
关键词密度 = (关键词出现次数 × 关键词字数) ÷ 总字数 × 100%

示例：
文章总字数：800字
"比亚迪"出现次数：10次
"比亚迪"字数：3字

密度 = (10 × 3) ÷ 800 × 100% = 3.75% ✅
```

### 密度问题诊断

| 密度 | 问题 | 解决方案 |
|------|------|----------|
| <1% | 关键词不足，搜索难触达 | 在标题、首段、小标题增加关键词 |
| 2-3% | ✅ 理想状态 | 保持 |
| >5% | 关键词堆砌，影响阅读 | 使用同义词替换，减少重复 |

## 关键词挖掘方法

### 1. 搜索框下拉

```
小红书搜索框：
输入"比亚迪" → 查看下拉推荐
- 比亚迪财报
- 比亚迪销量
- 比亚迪海豹
- 比亚迪秦plus

这些是用户真实搜索的长尾词
```

### 2. 相关笔记分析

```
1. 搜索核心词
2. 查看高赞笔记的：
   - 标题用词
   - 标签组合
   - 评论区高频词
```

### 3. 热点事件挖掘

```
1. 关注财经日历
   - 财报发布日期
   - 产品发布会
   - 政策发布

2. 结合热点生成关键词
   - "财报季" + 公司名 = "比亚迪财报季"
   - "降价" + 行业 = "新能源车降价"
```

### 4. 关键词组合矩阵

```
核心词 × 时间词 = 长尾词
比亚迪 × Q3 = 比亚迪Q3财报
新能源 × {年份} = {年份}新能源投资

核心词 × 动作词 = 长尾词
茅台 × 估值 = 茅台估值分析
小鹏 × 销量 = 小鹏销量数据

核心词 × 修饰词 = 长尾词
腾讯 × 深度 = 腾讯深度分析
白酒 × 投资 = 白酒投资逻辑
```
