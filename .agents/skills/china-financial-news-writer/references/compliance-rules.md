# 金融内容合规规则 (Compliance Rules)

## 核心原则

1. **不承诺收益**: 禁止任何形式的收益承诺
2. **不构成建议**: 明确声明不构成投资建议
3. **充分披露风险**: 提示投资风险
4. **数据来源透明**: 标注数据来源
5. **避免绝对化**: 不使用绝对化用语

---

## B站视频合规要求

### 内容规范

```
✅ 允许：
- 财报解读（客观分析）
- 行业分析（数据+观点）
- 投资知识科普
- 公司深度研究
- 风险提示

✅ 必须包含：
- 风险提示
- "仅供参考"声明
- 不构成投资建议
```

### 标题规范

```
❌ 禁止：
- "必涨"、"100%"、"稳赚"
- 过度标题党
- 虚假信息

✅ 允许：
- "看好"、"关注"、"值得期待"
- 客观描述+观点
- 数据支撑的结论
```

### 合规模板

```markdown
[视频内容]

---

⚠️ 风险提示：本文仅代表个人观点，不构成任何投资建议。
投资有风险，入市需谨慎！

#财经 #财报解读 #投资理财
```

## 内容审查清单

### 发布前必查

```
□ 是否包含风险提示
□ 是否声明"仅供参考"
□ 是否避免直接买入建议
□ 是否避免收益承诺
□ 标题是否合规（无绝对化用语）
□ 观点是否有数据支撑
□ 数据来源是否标注
```

## 敏感内容处理

### 发现敏感词时的处理流程

```
1. 标记敏感词
   ├── 记录词汇和位置
   └── 判断风险等级

2. 评估上下文
   ├── 是否可以替换
   ├── 是否可以删除
   └── 是否需要重写

3. 提供替代方案
   ├── 给出安全替换词
   └── 给出改写建议

4. 重新检查
   ├── 确认替换后无问题
   └── 确认语境通顺

5. 添加必要声明
   ├── 风险提示
   └── 免责声明
```

### 不同风险等级的处理

| 风险等级 | 处理方式 | 示例 |
|----------|----------|------|
| 🔴 高危 | 必须删除或重写 | "必涨"→"值得关注" |
| 🟠 中危 | 建议替换或加声明 | "推荐"→"看好"+风险提示 |
| 🟡 低危 | 可保留，建议加声明 | "收益"→+风险提示 |
| 🟢 安全 | 正常发布 | 客观数据分析 |

## 特殊场景合规

### 财报分析

```
✅ 可以说：
- "营收超预期X%"
- "毛利率环比提升X个百分点"
- "管理层指引..."
- "从数据看..."

❌ 不能说：
- "财报超预期，必涨"
- "业绩这么好，赶紧买"
- "100%会涨"
```

### 行业分析

```
✅ 可以说：
- "行业增速XX%"
- "市场份额XX%"
- "竞争格局分析..."
- "受益于..."

❌ 不能说：
- "这个板块必涨"
- "强烈推荐买入"
- "稳赚的机会"
```

### 政策解读

```
✅ 可以说：
- "政策要点..."
- "对行业的影响..."
- "受益方向..."
- "值得关注..."

❌ 不能说：
- "政策利好，必涨"
- "这是抄底机会"
- "满仓干"
```

## 数据引用规范

### 必须标注来源的数据

```
- 财务数据（营收、利润等）
- 市场数据（股价、成交量等）
- 行业数据（市场规模、增速等）
- 调研数据（用户调研、市场调研等）
```

### 数据来源标注格式

```
文中标注：
"根据公司财报，Q3营收为XX亿元"
"Bloomberg数据显示..."

图表标注：
"Source: 公司财报，XX证券"
"Source: Bloomberg（截至{年份}年{月}月{日}日）"

文末标注：
"数据来源：公司财报、Bloomberg、Wind"
```

### 禁止的数据使用

```
❌ 禁止：
- 使用来源不明的数据
- 伪造或篡改数据
- 断章取义引用数据
- 过期数据不加时间说明
```

## 利益冲突披露

### 需要披露的情况

```
1. 持有相关股票
   "披露：作者持有XX公司股票"

2. 近期交易相关股票
   "披露：作者近30日内交易过XX公司股票"

3. 提供付费服务
   "披露：作者提供付费投资咨询服务"

4. 与公司有商业合作
   "披露：本文为XX公司约稿"
```

### 披露位置

```
- 视频简介末尾或置顶评论
- 醒目但不影响观看
- 格式统一
```
