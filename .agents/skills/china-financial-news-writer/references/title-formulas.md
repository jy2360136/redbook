# 金融文章标题公式库 (Title Formulas for Financial Content)

## 核心原则

好标题 = **关键词** + **价值预期** + **时效性/独特性**

---

## 八大爆款标题公式（金融版）

### 1️⃣ 数字具象型
**公式**: 数字 + 核心结论 + 时间范围

```
✅ "比亚迪Q3营收破2000亿，三个数据最关键"
✅ "小鹏MONA上市72小时：订单破3万的背后"
✅ "腾讯Q3财报：游戏业务增长12%，广告更亮眼"
✅ "5家新能源车企销量对比：谁在加速谁在掉队"
✅ "茅台Q3净赚150亿，毛利率稳定在91%"

❌ "比亚迪Q3财报分析"（无信息量）
❌ "关于腾讯的一些看法"（过于笼统）
```

**为什么有效**: 数字给人具体、可量化的感觉，降低阅读门槛

---

### 2️⃣ 事件+观点型
**公式**: 重大事件 + 核心观点/判断

```
✅ "央行降准落地：对股市的影响比想象更大"
✅ "比亚迪销量破50万：价格战的赢家出现了"
✅ "小鹏MONA发布：15万级市场的格局要变了"
✅ "茅台提价20%：白酒龙头的底气从哪来"
✅ "腾讯回购千亿：信号比金额更重要"

❌ "央行降准了"（缺少观点）
❌ "比亚迪销量分析"（缺少核心结论）
```

**为什么有效**: 事件吸引关注，观点提供价值

---

### 3️⃣ 对比反差型
**公式**: Before/After 或 A vs B

```
✅ "从亏损百亿到盈利：蔚来做对了什么"
✅ "理想vs问界：新势力销量王的争夺战"
✅ "比亚迪销量第一，为什么股价却不涨？"
✅ "同样是降价，为什么比亚迪能赢特斯拉不能"
✅ "茅台和五粮液：差距正在缩小还是拉大"

❌ "蔚来分析"（缺少对比张力）
```

**为什么有效**: 对比产生冲突感，引发好奇

---

### 4️⃣ 提问悬念型
**公式**: 问题 + 暗示答案有价值

```
✅ "比亚迪月销50万，为什么机构还在卖？"
✅ "小鹏能靠MONA翻身吗？从数据找答案"
✅ "新能源车降价潮：谁在裸泳谁在抄底"
✅ "茅台的护城河，到底有多深？"
✅ "腾讯的游戏业务，真的见底了吗"

❌ "比亚迪分析"（没有悬念）
❌ "新能源车怎么样"（问题太宽泛）
```

**为什么有效**: 问题制造信息缺口，驱动点击

---

### 5️⃣ 身份权威型
**公式**: 身份/角色 + 独家洞察

```
✅ "券商分析师：看财报最该关注的5个指标"
✅ "基金经理：为什么我现在不买新能源"
✅ "十年老股民：这些财报信号最危险"
✅ "行业研究员：比亚迪的真正护城河在哪"
✅ "PE投资人：怎么看新能源车的估值"

❌ "财报分析方法"（缺少身份背书）
```

**为什么有效**: 专业身份增加可信度

---

### 6️⃣ 情绪共鸣型
**公式**: 情绪词 + 核心内容

```
✅ "终于！新能源车的价格战要结束了？"
✅ "没想到！小鹏MONA配置比预想更顶"
✅ "财报季来了，这3家公司要小心"
✅ "重磅！央行出手，A股要变天"
✅ "遗憾！又一家新势力撑不住了"

❌ "新能源车价格分析"（没有情绪）
```

**为什么有效**: 情绪词吸引注意，增加代入感

---

### 7️⃣ 场景代入型
**公式**: 具体场景 + 解决方案/分析

```
✅ "财报季来了！小白也能看懂的解读指南"
✅ "想买新能源股票？先看这3个指标"
✅ "茅台跌破1500，是抄底还是陷阱"
✅ "准备入场？这份新能源车投资清单请收好"
✅ "{年份}年买什么？从Q3财报找答案"

❌ "投资建议"（没有场景感）
```

**为什么有效**: 场景具体，读者容易代入

---

### 8️⃣ 清单合集型
**公式**: 主题 + 合集/盘点/排名

```
✅ "{年份}年Q3新能源车企财报全盘点"
✅ "{月}月新能源车销量排行榜：格局变了"
✅ "5只被低估的消费股，值得关注"
✅ "白酒行业Q3业绩对比：谁是真龙头"
✅ "券商金股{月}月名单：这些票被集中推荐"

❌ "新能源车销量"（缺少合集属性）
```

**为什么有效**: 信息密度高，收藏价值高

## 平台适配规则

### 小红书标题（15-20字）

```
特征：
- 情绪词更重
- emoji适度使用
- 口语化表达
- 制造好奇

公式：[情绪词] + [核心信息] + [emoji]

示例：
"救命！比亚迪这份财报太顶了🔥"
"小鹏要翻身了？MONA销量有点东西📈"
"财报季！小白也能看懂的解读👇"

避免：
- 过于专业的术语
- 太长的句子
- 缺少情绪词
```

### 公众号标题（20-30字）

```
特征：
- 信息量更大
- 时效性强调
- 专业感适度
- 观点明确

公式：[公司/事件] + [核心数据/变化] + [价值预告]

示例：
"比亚迪Q3营收破2000亿，这三个信号最重要"
"小鹏MONA发布：15万级市场的价格战才刚开始"
"央行降准落地，对A股影响几何？"

避免：
- 标题党但内容不符
- 过度情绪化
- 缺少核心信息
```

### 研报标题（30-40字）

```
特征：
- 公司全称+代码
- 事件描述
- 核心观点
- 评级

公式：[公司名]([代码])：[事件] - [核心观点] - [评级]

示例：
"比亚迪(002594)：Q3营收超预期，盈利能力持续提升 - 买入"
"小鹏汽车(XPEV)：MONA M03发布，下沉市场战略落地 - 增持"

避免：
- 情绪词
- 夸大表述
- 模糊观点
```

## 标题优化检查清单

### 必须包含 ✅

- [ ] 核心关键词（公司名/行业名）
- [ ] 价值点（读者能获得什么）
- [ ] 时效性（如果适用）
- [ ] 观点或结论（至少暗示）

### 避免 ❌

- [ ] 纯品牌名做标题
- [ ] 过于笼统无具体信息
- [ ] 绝对化用语（最好、第一、100%）
- [ ] 标题党但内容不符
- [ ] 过度情绪化（针对公众号/研报）
- [ ] 错别字和数据错误

## 敏感词规避

### 金融标题中的禁忌

| 禁忌词 | 替代方案 |
|--------|----------|
| 必涨、一定涨 | 有望上涨、可能受益 |
| 稳赚、保本 | 值得关注、具有配置价值 |
| 内幕消息 | 市场传闻、行业动态 |
| 100%、绝对 | 较高概率、较大可能 |
| 推荐、强烈推荐 | 关注、看好 |
| 最佳、最好 | 优质、领先 |

### 合规表述示例

```
❌ "这只股票必涨！强烈推荐买入"
✅ "这家公司值得关注，成长性较好"

❌ "100%赚钱的机会来了"
✅ "从历史数据看，这类机会值得关注"

❌ "内幕：XX公司要大利好"
✅ "市场预期XX公司可能有积极变化"
```
