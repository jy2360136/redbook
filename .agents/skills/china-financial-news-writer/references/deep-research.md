# 深度情报搜集指南 (Deep Intelligence Gathering)

## 概述

本指南用于在金融新闻写作流程中，对特定公司/事件进行**全网深度调研**，确保分析报告的完备性、准确性和洞察力。

**适用场景**：
- 重大财报事件（如"本田利润暴跌 90%"）
- 突发负面新闻（如"某公司被立案调查"）
- 行业重大变化（如"政策突变"）
- 需要深度分析的投资标的

---

## 调研框架 (6 维情报网)

### 维度 1: 新闻媒体报道 (News Media)

**目标**：获取权威、及时的事件报道

| 类型 | 来源 | 搜索策略 |
|------|------|----------|
| **主流财经** | 虎嗅、36 氪、财新、界面 | 公司名 + 事件关键词 |
| **门户财经** | 新浪财经、网易财经、腾讯财经 | 百度新闻搜索 |
| **垂直媒体** | 第一财经、证券时报、中国基金报 | 官网搜索 + RSS |
| **海外媒体** | WSJ 中文、FT 中文、Bloomberg | 英文 + 中文双向搜索 |

**搜索关键词构建**：
```
基础：{公司名} + {事件核心词}
  例：本田 + 利润暴跌

扩展：{公司名} + {事件} + 原因/分析/解读
  例：本田 + 利润 + 原因分析

深度：{公司名} + {细分问题}
  例：本田 + 中国销量 + 下滑
```

**信息提取要点**：
- [ ] 事件发布时间
- [ ] 核心数据（利润、营收、同比等）
- [ ] 官方回应/公告
- [ ] 分析师初步评论

---

### 维度 2: 视频平台内容 (Video Content)

**目标**：获取可视化解读、专家观点、情绪感知

| 平台 | 搜索策略 | 价值点 |
|------|----------|--------|
| **B 站** | 公司名 + 财经/分析 | UP 主深度解读、弹幕情绪 |
| **抖音** | 公司名 + 热点话题 | 短视频传播、大众认知 |
| **西瓜视频** | 财经频道搜索 | 中长视频分析 |
| **YouTube** | 公司英文名 + analysis | 海外视角 |

**B 站搜索示例**：
```
搜索词：本田 利润 暴跌
筛选：发布时间（一周内）、时长（10 分钟 +）、播放量（1 万+）

关注：
- 财经区 UP 主（如：小lin 说、巫师财经）
- 汽车行业 UP 主
- 弹幕和评论区的用户情绪
```

**信息提取要点**：
- [ ] UP 主核心观点
- [ ] 视频提供的独特数据/视角
- [ ] 评论区高频关键词
- [ ] 情绪倾向（正面/负面/中性）

### 维度 3: 社交媒体讨论 (Social Media)

**目标**：感知公众情绪、发现隐藏信息

#### 3.1 微信公众号（搜狗微信搜索）

**搜索策略**：
```
site:weixin.qq.com {公司名} {事件}
或直接用搜狗微信搜索：https://weixin.sogou.com/

筛选：
- 发布时间（近 7 天）
- 公众号类型（财经类优先）
- 阅读量（1 万 + 优先）
```

**关注账号类型**：
- 财经自媒体（如：饭统戴老板、远川研究所）
- 行业垂直号（如：汽车商业评论）
- 券商研究所官方号
- 公司官方公众号

#### 3.2 微博

**搜索策略**：
```
{公司名} + {事件}
筛选：实时、热门

关注：
- 热搜话题（如有）
- 财经大 V 观点
- 公司官方微博回应
```

#### 3.3 小红书

**搜索策略**：
```
{公司名} + 投资/理财/分析

关注：
- 个人投资者的真实感受
- 是否有内部人士透露信息
- 评论区讨论
```

**信息提取要点**：
- [ ] 公众情绪倾向
- [ ] 是否有独家信息源
- [ ] 高频讨论话题
- [ ] 是否有谣言需要辟谣

### 维度 4: 投资社区/论坛 (Investment Communities)

**目标**：获取深度分析、专业观点、散户情绪

#### 4.1 雪球（最重要）

**搜索策略**：
```
{公司名} 或 {股票代码}
筛选：热门、最新、深度

重点关注：
- 大 V 用户的长文分析
- 评论区专业讨论
- 历史文章（了解背景）
```

**信息提取要点**：
- [ ] 深度分析文章的核心逻辑
- [ ] 数据支撑是否充分
- [ ] 多空观点对比
- [ ] 高赞评论的洞察

#### 4.2 东方财富股吧

**搜索策略**：
```
{股票代码} 或 {公司简称}

关注：
- 散户情绪（极度悲观/乐观可能是反向指标）
- 是否有内部消息流传
- 热门讨论话题
```

#### 4.3 知乎

**搜索策略**：
```
{公司名} + {事件} + 分析/评价

筛选：
- 高赞回答
- 专业背景答主（标注行业从业者的）
- 长文分析
```

**信息提取要点**：
- [ ] 专业从业者的 insider 视角
- [ ] 系统性分析框架
- [ ] 历史对比分析
- [ ] 行业横向对比

#### 4.4 其他社区

| 社区 | 特点 | 关注点 |
|------|------|--------|
| 老虎社区/富途牛牛 | 港股/美股投资者 | 海外投资者视角 |
| 同花顺论坛 | A 股散户 | 情绪指标 |
| 豆瓣理财小组 | 理财小白 | 大众认知 |

## 信息处理流程

### Step 1: 信息收集 (30-60 分钟)

```
1. 按 6 个维度依次搜索
2. 每个维度保存 3-5 个高价值链接
3. 记录关键数据、观点、时间戳
4. 使用表格或笔记工具整理
```

**推荐工具**：
- Notion/飞书文档（结构化整理）
- 浏览器书签（分类保存）
- 截图（重要图表、数据）

### Step 2: 信息验证 (15-30 分钟)

```
1. 交叉验证关键数据
   - 同一数据在多个来源是否一致
   - 官方数据 vs 媒体报道

2. 识别谣言/不实信息
   - 无来源的"内部消息"
   - 过度情绪化的表述
   - 与官方信息矛盾的

3. 标注信息来源可信度
   - ⭐⭐⭐ 官方/权威媒体
   - ⭐⭐ 知名财经自媒体/大 V
   - ⭐ 普通用户/匿名来源
```

### Step 3: 信息整合 (30-45 分钟)

```
1. 按时间线梳理事件
2. 归纳核心观点（支持/反对/中性）
3. 提取关键数据
4. 识别信息缺口（需要进一步搜索的）
```

## 注意事项

### 合规提醒
1. **不传播谣言**：未经证实的"内部消息"需标注
2. **注明出处**：所有引用数据、观点需标注来源
3. **避免荐股**：分析报告不构成投资建议
4. **风险提示**：必须包含风险提示

### 效率建议
1. **先官方后媒体**：优先获取一手信息
2. **先深度后广度**：先读几篇深度分析，再扩展
3. **设定时间限制**：避免过度搜索导致拖延
4. **善用搜索技巧**：site:、intitle:等高级搜索
