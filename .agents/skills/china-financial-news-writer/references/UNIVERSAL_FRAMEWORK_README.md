# 万能财经分析框架使用说明

## 🎯 框架定位
这是一个通用的企业危机事件分析框架，适用于任何公司的财务危机、战略失误、市场竞争等分析场景。

## 📋 框架特色

### 1. 全面性
- 涵盖财经分析的所有可能角度
- 从数据到故事，从现状到未来
- 适合不同层次的分析需求

### 2. 灵活性
- 模块化设计，可自由组合
- 平台适配指南，一键切换风格
- 内容可增可减，完全自定义

### 3. 实用性
- 提供具体的数据收集指引
- 包含可视化建议
- 附带平台适配方案

## 🔧 使用方法

### 第一步：事件诊断
1. 确定分析对象（具体公司）
2. 识别危机类型（财务/战略/市场/技术等）
3. 评估影响范围（行业/区域/全球）

### 第二步：模块选择
根据分析目标和平台特性，勾选相关模块：

**B站视频（10分钟）推荐组合**：
- 核心模块（1、2、3）必选
- 深度洞察（6、7）选1-2个
- 内容表达（9、10、11）必选
- 数据分析（4）简要

### 第三步：数据填充
针对选中的每个模块，收集相应数据：

**数据来源建议**：
- 财报数据：公司官网、交易所公告
- 市场数据：IDC、Counterpoint等机构报告
- 舆情数据：微博、知乎、雪球等平台
- 历史数据：Wind、Bloomberg等数据库

### 第四步：内容生成
使用AI工具基于选定模块生成具体内容：

**提示词模板**：
```
请基于以下框架为[公司名称]生成[平台类型]风格的[内容类型]：

1. 使用模块：[选中的模块编号]
2. 风格要求：[B站/小红书/公众号等]
3. 时长/字数：[10分钟/500字/2000字等]
4. 重点突出：[核心观点/数据支撑/情感共鸣等]
5. 互动设计：[投票/评论引导/下期预告等]
```

## 📈 应用场景

### 场景1：企业危机分析
- 利润暴跌、营收下滑
- 战略失误、项目失败
- 市场份额丢失

### 场景2：行业竞争分析
- 新兴竞争者冲击
- 技术路线变迁
- 市场格局重塑

### 场景3：投资价值分析
- 企业估值变化
- 行业趋势判断
- 投资风险提示

### 场景4：内容创作
- 视频脚本大纲
- 文章写作框架
- 社交媒体话题

## 🎨 平台适配指南

### B站风格特点
- **时长**：8-15分钟

- **语言**：口语化+专业术语解释
- **视觉**：图表+动画+对比
