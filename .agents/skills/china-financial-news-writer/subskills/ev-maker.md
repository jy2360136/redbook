## 财报分析框架

### 1. 销量分析 (30%)

```
分析要点：
- 季度交付量及环比变化
- 与指引对比
- 与竞品对比
- 市占率变化

销量数据来源：
- 公司公告（月度交付）
- 乘联会（批发/零售）
- 中汽协（行业数据）

常见问题：
Q: 批发和零售有什么区别？
A: 批发=厂商出货量，零售=终端销售量
```

### 2. 盈利能力分析 (25%)

```
分析要点：
- 汽车毛利率（剔除新能源积分）
- 单车毛利
- 单车净利润
- 费用率变化

关键公式：
单车毛利 = 汽车毛利 ÷ 交付量
单车净利 = 净利润 ÷ 交付量

行业标准：
- 毛利率>20%：优秀（比亚迪、理想）
- 毛利率15-20%：中等
- 毛利率<15%：承压
```

### 3. 现金流分析 (20%)

```
分析要点：
- 经营性现金流
- 自由现金流
- 现金储备
- 融资能力

生存能力评估：
- 现金储备 > 12个月烧钱：安全
- 现金储备 6-12个月烧钱：需关注
- 现金储备 < 6个月烧钱：危险
```

### 4. 技术与产品分析 (15%)

```
分析要点：
- 新车型规划
- 技术路线（纯电/增程/插混）
- 智驾能力
- 电池技术

智驾评估维度：
- 硬件配置（算力、传感器）
- 软件能力（NOA开通城市）
- 渗透率（选装比例）
```

### 5. 战略分析 (10%)

```
分析要点：
- 品牌定位
- 出海进展
- 子品牌战略
- 合作伙伴

出海关注点：
- 欧洲市场进展
- 东南亚市场
- 美国市场（受限）
```

## 行业分析框架

### 价格战分析

```
分析维度：
1. 降价幅度
   - 官方降价
   - 终端优惠
   - 权益变化

2. 降价原因
   - 清库存
   - 抢份额
   - 成本下降

3. 降价影响
   - 毛利率压力
   - 消费者观望
   - 二手车残值

写作示例：
"{当前年份}年新能源车价格战回顾：
- 比亚迪率先降价10%
- 特斯拉跟进
- 新势力被动应战
消费者笑了，但车企呢？"
```
