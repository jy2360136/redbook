# 金融券商写作子技能 (Financial Institution Writing Subskill)

## 适用公司

券商：中信证券、中金公司、华泰证券、东方财富、招商证券
银行：招商银行、宁波银行、平安银行、工商银行
保险：中国平安、中国人寿、中国太保

---

## 公司类型特征

### 业务特征
- 高度依赖市场环境
- 周期性明显
- 监管严格
- 资本密集

### 财报关注重点
| 指标 | 券商 | 银行 | 保险 |
|------|------|------|------|
| 核心指标 | 经纪市占率 | 净息差 | 新业务价值 |
| 盈利指标 | ROE | 不良率 | 内含价值 |
| 风险指标 | 杠杆率 | 拨备覆盖率 | 偿付能力 |
| 业务指标 | 两融余额 | 贷款增速 | 保费收入 |

---

## 券商分析框架

### 核心业务分析

```
1. 经纪业务 (25%)
   - 股基交易市占率
   - 佣金率变化
   - 财富管理转型

2. 投行业务 (20%)
   - IPO承销规模/数量
   - 再融资规模
   - 债券承销规模
   - 行业排名

3. 资管业务 (15%)
   - 资产管理规模
   - 主动管理占比
   - 管理费率

4. 自营业务 (25%)
   - 投资规模
   - 投资收益率
   - 资产配置

5. 资本中介 (15%)
   - 两融余额
   - 质押规模
   - 利差收入
```

### 写作框架

```
券商财报分析结构：

1. 整体业绩
   - 营收及增速
   - 净利润及增速
   - ROE变化

2. 分业务分析
   - 经纪：交易额市占率
   - 投行：承销排名
   - 资管：规模变化
   - 自营：投资收益
   - 资本中介：两融规模

3. 市场环境影响
   - A股成交额
   - IPO发行节奏
   - 债券市场表现

4. 监管政策影响
   - 佣金率政策
   - 再融资政策
   - 衍生品政策

5. 估值分析
   - PB估值
   - 历史分位

6. 投资建议
   - 市场判断
   - 个股选择
```

### 东方财富写作示例

```
公众号标题：
"东方财富Q3财报：市占率继续提升，基金销售回暖"

小红书标题：
"东方财富Q3营收XX亿！互联网券商一哥🔥"

正文框架：
"东方财富Q3财报发了，互联网券商一哥！

📊 核心数据：
• 营收：XX亿，同比+X%
• 净利润：XX亿，同比+X%

📈 业务亮点：
1️⃣ 股基交易市占率提升至X%
2️⃣ 两融余额市占率X%
3️⃣ 基金销售额XX亿（回暖）

⚠️ 关注：
- A股成交额
- 基金销售恢复节奏

东财的核心优势：
• 流量成本低
• 用户粘性高
• 变现能力强

你们用东方财富炒股吗？👇

#东方财富 #券商 #财报"
```

   - 拨备覆盖率

## 市场环境分析

### A股市场对券商影响

```
牛市特征：
- 成交额放大 → 经纪收入大增
- IPO加速 → 投行收入增加
- 自营收益提升
- 两融余额上升

熊市特征：
- 成交额萎缩 → 经纪收入下降
- IPO放缓 → 投行收入承压
- 自营可能亏损
- 两融余额下降

写作示例：
"A股日成交额突破万亿，券商股要起飞？
从历史看，成交额破万亿时，券商板块平均涨幅XX%"
```

### 利率对银行影响

```
利率下行影响：
- 息差收窄（负面）
- 债券浮盈（正面）
- 信贷需求提振（正面）

利率上行影响：
- 息差可能扩大（正面）
- 债券浮亏（负面）
- 信贷需求压制（负面）

写作示例：
"LPR下调对银行影响几何？
息差承压，但资产质量可能改善"
```

## 估值分析

### 常用估值方法

```
券商：
- PB（市净率）：0.8-1.5倍
- 关注历史分位

银行：
- PB（市净率）：0.5-0.8倍
- 股息率：5-7%

保险：
- P/EV（股价/内含价值）：0.5-1.0倍
- P/NBV
```

### 估值写作示例

```
"券商股估值到什么位置了？

当前PB：X.XX倍
历史最低：X.XX倍
历史最高：X.XX倍
历史分位：XX%

从估值看，券商股处于历史[低位/中位/高位]

⚠️ 估值低不代表会涨，还需看市场环境"
```
