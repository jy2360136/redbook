## 财报分析框架

### 1. 用户指标分析 (25%)

```
分析要点：
- DAU/MAU变化
- 用户时长变化
- 用户画像变化
- 获客成本变化

常见问题：
Q: 用户增长放缓怎么办？
A: 分析ARPU提升空间、新用户来源、海外拓展
```

### 2. 收入结构分析 (25%)

```
分析要点：
- 分业务收入占比
- 各业务增速
- 变现率变化
- 收入质量

常见问题：
Q: 如何判断收入质量？
A: 看现金收入比、应收账款、递延收入
```

### 3. 利润率分析 (20%)

```
分析要点：
- 毛利率变化
- 费用率变化
- 各业务利润贡献
- Non-GAAP vs GAAP

常见问题：
Q: 为什么看Non-GAAP？
A: 排除股权激励、投资损益等非经营因素
```

### 4. 新业务分析 (15%)

```
分析要点：
- 新业务收入占比
- 新业务亏损情况
- 新业务战略意义
- 管理层表态

常见业务：
- 腾讯：视频号、企业服务
- 阿里：云业务、国际商业
- 美团：即时零售
- 京东：物流、健康
```

### 5. 监管与竞争 (15%)

```
分析要点：
- 反垄断监管
- 数据安全合规
- 竞争格局变化
- 行业政策

常见监管影响：
- 游戏版号
- 电商合规
- 数据安全
- 金融监管
```

## 小红书写作风格

### 标题公式

```
情绪型：
"腾讯财报来了！游戏业务终于回暖🔥"
"救命！阿里这份财报有点东西"

数字型：
"腾讯Q3财报：3个数据最重要"
"阿里云收入增速回升到7%！"

提问型：
"腾讯股价为什么不涨？财报里有答案"
"美团vs抖音，谁赢了？"
```

### 正文结构

```
开头（50字内）：
[公司]Q[季度]财报出炉，[核心结论]！

核心数据（emoji+短句）：
📊 营收：[数字]
💰 利润：[数字]
📱 用户：[数字]

亮点解读：
1️⃣ [亮点1]
2️⃣ [亮点2]
3️⃣ [亮点3]

隐忧：
⚠️ [隐忧]

结尾：
[互动问题]

#[公司名] #财报解读 #科技股
```

## 研报写作风格

### 标题格式

```
[公司名]([代码])：[事件] - [评级]

示例：
腾讯控股(0700.HK)：游戏回暖，广告超预期 - 买入
阿里巴巴(BABA)：淘天企稳，云业务盈利 - 增持
```

### 分析框架

```
1. 投资要点
2. 业绩概述
3. 分业务分析
   3.1 核心业务
   3.2 新兴业务
4. 盈利能力分析
5. 估值与建议
6. 风险提示
```
