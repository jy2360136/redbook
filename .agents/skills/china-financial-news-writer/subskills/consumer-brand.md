# 消费品牌写作子技能 (Consumer Brand Writing Subskill)

## 适用公司

白酒：贵州茅台、五粮液、泸州老窖、山西汾酒
运动服饰：安踏、李宁、特步
美妆护肤：珀莱雅、贝泰妮、上海家化
食品饮料：伊利、蒙牛、农夫山泉
餐饮：瑞幸、海底捞、九毛九

---

## 公司类型特征

### 业务特征
- 品牌力为核心
- 渠道为王
- 消费趋势敏感
- 季节性明显

### 财报关注重点
| 指标 | 重要性 | 说明 |
|------|--------|------|
| 收入增速 | ⭐⭐⭐⭐⭐ | 增长能力 |
| 毛利率 | ⭐⭐⭐⭐⭐ | 品牌力体现 |
| 同店增长 | ⭐⭐⭐⭐ | 渠道质量 |
| 库销比 | ⭐⭐⭐⭐ | 库存健康度 |
| 经营现金流 | ⭐⭐⭐⭐ | 盈利质量 |

## 行业分类写作要点

### 白酒行业

#### 核心指标

```
必须关注：
- 营收及增速
- 净利润及净利率
- 毛利率变化
- 分产品收入（高端/中端）
- 渠道结构（直销/批发）
- 预收款/合同负债

可选关注：
- 省内/省外占比
- 经销商数量变化
- 吨价变化
```

#### 写作框架

```
财报分析结构：

1. 整体业绩
   - 营收增速
   - 利润增速
   - 与行业对比

2. 产品结构
   - 高端产品增速
   - 中端产品表现
   - 产品升级趋势

3. 渠道分析
   - 直销占比
   - 经销商变化
   - 电商表现

4. 区域分析
   - 省内/省外
   - 重点市场

5. 库存与预收
   - 合同负债变化
   - 渠道库存情况

6. 行业对比
   - 与竞品对比
   - 行业排名变化
```

#### 茅台写作示例

```
公众号标题：
"茅台Q3财报：营收增速15%，系列酒表现亮眼"

小红书标题：
"茅台Q3又赚了XX亿！白酒一哥有多稳🔥"

正文框架：
"茅台Q3财报出炉，用一个字总结：稳！

📊 核心数据：
• 营收：XX亿，同比+X%
• 净利润：XX亿，同比+X%
• 毛利率：XX%，基本持平

✨ 亮点：
1️⃣ 茅台酒收入稳健增长
2️⃣ 系列酒增速超预期
3️⃣ 直销占比继续提升

⚠️ 关注：
- 消费复苏节奏
- 价格政策

白酒龙头还是稳！
你们觉得茅台值得投资吗？👇

#茅台 #白酒 #财报解读"
```

## 消费趋势分析

### 常见消费趋势

```
1. 消费降级/理性消费
   - 关注性价比
   - 平替产品
   - 折扣渠道

2. 国潮崛起
   - 国货品牌力提升
   - 文化自信
   - 年轻人偏好

3. 健康化
   - 低糖/低卡
   - 天然成分
   - 功能性

4. 个性化
   - 小众品牌
   - 定制化
   - 圈层消费
```

### 趋势写作框架

```
"消费趋势观察：[趋势名称]

现象描述：
[描述当前消费现象]

数据支撑：
- [数据1]
- [数据2]

影响分析：
• 受益品牌：[品牌]
• 受损品牌：[品牌]

投资建议：
[建议]（仅供参考）
"
```

## 小红书写作技巧

### 标题公式

```
财报类：
"茅台Q3又赚了XX亿！白酒一哥有多稳🔥"
"安踏营收破XX亿！国货运动一哥实锤"

行业类：
"白酒股集体大跌，发生了什么？"
"国货美妆崛起！这些品牌值得关汪"

消费类：
"为什么年轻人都在喝瑞幸？"
"消费降级时代，这些品牌火了"
```

### 互动设计

```
评论引导：
"你们平时买安踏还是李宁？"
"喝瑞幸的举手🙋"
"茅台和五粮液你选哪个？"

"评论区投票：
🅰️ 茅台
🅱️ 五粮液
🅾️ 都不喝"
```

## 季节性分析

### 白酒

```
旺季：
- Q1：春节备货
- Q3：中秋国庆

淡季：
- Q2：传统淡季
- Q4：控货挺价

写作注意：
- 关注节前备货情况
- 关注价格走势
- 关注库存水平
```

### 运动服饰

```
旺季：
- Q1：春节+开年
- Q3：开学季+换季
- Q4：双十一+冬装

淡季：
- Q2：相对平淡

写作注意：
- 关注换季清库存
- 关注大促表现
- 关注新品发布
```

### 餐饮

```
旺季：
- Q1：春节
- Q2：五一
- Q3：暑期
- Q4：国庆+年底

写作注意：
- 关注节假日表现
- 关注同店增长
- 关注翻台率
```
