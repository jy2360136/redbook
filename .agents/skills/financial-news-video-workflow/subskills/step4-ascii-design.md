# Step 4: 布局设计

## 执行流程

### 4.1 读取输入

读取之前步骤的输出：
```
output/pipeline_output_xxx/manuscript/script_final.md    # 文案
output/pipeline_output_xxx/picture/prompts.json          # 配图信息
output/pipeline_output_xxx/picture/images/               # 图片文件
```

### 4.2 分镜规划

根据文案和图片数量，规划分镜：

**分镜规则**：
1. **开场**：封面图 + 标题（5-10秒）
2. **主体**：按文案结构分镜，每个观点一页
3. **数据展示**：数据图 + 关键数字
4. **过渡**：使用简单的淡入淡出
5. **结尾**：总结图 + 互动引导

**时长分配参考**：
| 内容类型 | 建议时长 |
|----------|----------|
| 封面/开场 | 5-10秒 |
| 核心观点 | 每条30-60秒 |
| 数据展示 | 每张15-30秒 |
| 过渡动画 | 1-2秒 |
| 结尾 | 10-15秒 |

### 4.4 用户调整确认

**暂停等待**：
- 展示 ASCII 草图
- 用户可以调整布局

**用户操作**：
```
Agent: 布局草图已生成！
       📄 位置：ascii_draft/layout_v1.txt

       分镜预览：
       • 总时长：约 12 分钟
       • 幻灯片数量：15 张
       • 配图数量：8 张

       需要调整吗？
       - 输入"修改 幻灯片02"来调整特定幻灯片
       - 输入"增加 数据展示"来插入新幻灯片
       - 输入"确认"继续下一步

用户: 修改 幻灯片02，把时长改成45秒
Agent: [修改草图] ...已更新幻灯片02

用户: 确认
Agent: [保存定稿] layout_v1.txt → layout_final.txt
       Step 4 完成！输入"继续"进入 Step 5
```
