# 中英双语图片提示词模板

## 提示词结构

### 基础模板

```
中文：[主体描述]，[场景/背景]，[风格]，[色调]
English: [Subject description], [Scene/Background], [Style], [Color tone], professional, 16:9
```

### 完整模板

```
中文：[主体]，[动作/状态]，[环境/场景]，[光线]，[氛围]，[风格]，[视角]，[画质]
English: [Subject], [Action/State], [Environment/Scene], [Lighting], [Mood], [Style], [View angle], [Quality], professional, 16:9
```

## 质量参数（可选）

如果生图平台支持，可以添加以下参数：

```
--ar 16:9          # 宽高比
--v 6              # 版本（Midjourney）
--q 2              # 质量
--s 750            # 风格化程度
--style raw        # 风格（Midjourney）
```

## 注意事项

1. **避免版权问题**：不要使用真实品牌Logo、真实人物面孔
2. **保持一致性**：同一视频的图片风格应保持一致
3. **适配字幕**：图片上方/下方预留字幕空间
4. **分辨率要求**：建议 1920x1080 或更高
