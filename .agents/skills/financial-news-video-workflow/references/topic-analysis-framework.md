# 选题分析框架

## 分析目标

从爬取的新闻中筛选出最适合制作深度视频的选题。

---

## 分析维度

### 1. 热点聚合度（权重 30%）

**计算方法**：
- 统计同一事件在不同媒体出现的次数
- 不同媒体权重：虎嗅(1.5) > 36氪(1.3) > 钛媒体(1.0) > 界面(1.0)

**评分标准**：
| 出现次数 | 评分 |
|----------|------|
| 1次 | 20分 |
| 2次 | 40分 |
| 3次 | 60分 |
| 4次 | 80分 |
| 5次+ | 100分 |

### 2. 时效性（权重 20%）

**计算方法**：
- 根据发布时间距离现在的时长

**评分标准**：
| 时间范围 | 评分 |
|----------|------|
| 24小时内 | 100分 |
| 48小时内 | 80分 |
| 72小时内 | 60分 |
| 1周内 | 40分 |
| 1周以上 | 20分 |

### 3. 影响力（权重 20%）

**计算方法**：
- 媒体权重 × 互动数据（如有）

**评分标准**：
| 影响力等级 | 评分 |
|------------|------|
| 头条新闻 + 多媒体报道 | 100分 |
| 首页推荐 | 80分 |
| 普通报道 | 60分 |
| 边缘新闻 | 40分 |

### 4. 视频适配度（权重 30%）

**评估因素**：

#### 4.1 数据丰富度
- 有明确的数字、百分比、排名：+20分
- 有可可视化的数据（图表潜力）：+20分
- 有历史对比数据：+10分

#### 4.2 故事性
- 有戏剧性转折：+20分
- 有人物故事：+15分
- 有历史背景：+10分

#### 4.3 复杂度适中
- 过于简单（5分钟能讲完）：-20分
- 过于复杂（需要30分钟+）：-10分
- 适中（8-15分钟）：+20分

#### 4.4 视觉潜力
- 有配图可能性：+10分
- 有视频素材可能性：+10分

---

## 综合评分公式

```
综合评分 = 热点聚合度 × 0.30
         + 时效性 × 0.20
         + 影响力 × 0.20
         + 视频适配度 × 0.30
```

---

## 选题类型判断

### 财报分析型
**关键词**：财报、Q1/Q2/Q3/Q4、营收、利润、同比增长

**适合原因**：
- 数据丰富，可视化空间大
- 有历史对比，故事性强
- 关注度高，时效性强

**切入点建议**：
- 数据冲击开场
- 历史对比展开
- 未来展望收尾

### 产品发布型
**关键词**：发布、上市、新品、交付、发布会

**适合原因**：
- 有明确的事件节点
- 视觉素材丰富
- 用户关注度高

**切入点建议**：
- 产品亮点开场
- 竞品对比分析
- 市场影响评估

### 人事变动型
**关键词**：离职、上任、高管、CEO、创始人

**适合原因**：
- 故事性强
- 人物背景可挖掘
- 对公司影响可分析

**切入点建议**：
- 人物背景开场
- 变动原因分析
- 对公司影响评估

### 政策影响型
**关键词**：政策、监管、新规、补贴、禁令

**适合原因**：
- 影响范围广
- 解读需求大
- 后续发展可预测

**切入点建议**：
- 政策内容概述
- 影响范围分析
- 应对策略建议

### 市场事件型
**关键词**：股价、市值、涨跌、暴跌、暴涨

**适合原因**：
- 数据可视化空间大
- 原因分析有价值
- 历史对比有趣

**切入点建议**：
- 数据冲击开场
- 原因深度分析
- 后续走势预测

---

## 排除标准

以下类型的新闻**不建议**选作视频选题：

1. **纯信息类**：没有分析空间，只是信息传达
2. **过于敏感**：涉及政治敏感话题
3. **时效已过**：超过1周的新闻，除非有深度分析价值
4. **过于专业**：需要专业背景才能理解的技术话题
5. **缺乏素材**：没有配图可能性的抽象话题
