# 事件新闻写作模板

**适用场景**：非财报类新闻，如产品发布、人事变动、政策影响、市场事件等

---

## 视频文案结构（8-15分钟）

### 一、开场钩子（30秒）

**目标**：在5秒内抓住观众注意力

**开场方式**：

1. **数据冲击型**
   ```
   50万辆！这是比亚迪今年Q1交出的成绩单。
   如果把时间倒回3年前，没有人能想到...
   ```

2. **悬念设置型**
   ```
   就在昨天，一家千亿市值的公司CEO突然宣布离职。
   这背后到底发生了什么？
   ```

3. **热点切入型**
   ```
   小米SU7正式交付了，雷军亲自给第一位车主开车门。
   但今天我想聊的不是车，而是...
   ```

4. **对比反差型**
   ```
   3年前，它还被嘲笑是"PPT造车"。
   3年后，它已经成了全球新能源销冠。
   ```

---

### 二、事件概述（1-2分钟）

**要素**：
- 时间：什么时候发生的
- 主体：谁、什么公司
- 事件：发生了什么
- 结果：有什么影响

**模板**：
```
[时间]，[公司/人物]正式宣布：[核心事件]。

这一消息迅速引发市场关注，[股价/销量/舆论]反应如何。

具体来看，[详细数据或事实]。
```

**示例**：
```
4月3日，小米汽车宣布SU7正式开始交付。
首批交付在北京、上海、深圳等7个城市同步启动，
雷军亲自为首批车主交车。

截至当天下午，小米SU7的锁单量已经突破10万台，
这一数字甚至超过了Model 3同期表现。
```

---

### 三、深度分析（5-8分钟）

**分析框架**（根据事件类型选择）：

#### 产品发布型

```
### 1. 产品力分析（2分钟）
- 核心卖点是什么？
- 与竞品相比如何？
- 技术亮点在哪里？

### 2. 市场定位（2分钟）
- 目标用户是谁？
- 定价策略如何？
- 差异化竞争点？

### 3. 商业模式（2分钟）
- 盈利空间多大？
- 规模化难点在哪？
- 长期护城河？
```

#### 人事变动型

```
### 1. 人物背景（2分钟）
- 谁离职/上任？
- 过往履历如何？
- 主要贡献是什么？

### 2. 变动原因（2分钟）
- 官方说法是什么？
- 市场猜测有哪些？
- 真实原因可能是？

### 3. 影响评估（2分钟）
- 对公司战略影响
- 对股价影响
- 对行业格局影响
```

#### 政策影响型

```
### 1. 政策解读（2分钟）
- 政策内容是什么？
- 为什么出台这个政策？
- 政策意图是什么？

### 2. 影响范围（2分钟）
- 哪些企业受影响？
- 影响程度如何？
- 时间节奏怎样？

### 3. 应对策略（2分钟）
- 企业如何应对？
- 行业如何调整？
- 投资者该如何决策？
```

#### 市场事件型

```
### 1. 事件复盘（2分钟）
- 发生了什么？
- 直接原因是什么？
- 深层原因是什么？

### 2. 历史对比（2分钟）
- 类似事件有哪些？
- 当时怎么处理的？
- 结果如何？

### 3. 后续预判（2分钟）
- 短期影响是什么？
- 中长期影响是什么？
- 关键观察点在哪？
```

---

### 四、历史对比（2-3分钟）

**目标**：提供更广阔的视角，增加内容深度

**对比方式**：

1. **纵向对比**：与自身历史对比
   ```
   回到2021年，比亚迪当时的月销量还只有...
   三年过去，这个数字翻了X倍。
   ```

2. **横向对比**：与竞品对比
   ```
   同期特斯拉中国的销量是...
   蔚来、小鹏、理想分别是...
   比亚迪一家的销量，超过了这三家的总和。
   ```

3. **历史案例**：类似的历史事件
   ```
   这让人想起2014年的小米...
   当年小米也是这样...
   结果后来...
   ```

---

### 五、未来展望（1-2分钟）

**目标**：给观众留下思考，增加互动

**展望角度**：

1. **趋势判断**
   ```
   从这个数据来看，我认为...
   未来一年，行业可能会...
   ```

2. **关键观察点**
   ```
   接下来我们要关注三个关键点：
   一是...，二是...，三是...
   ```

3. **风险提示**
   ```
   但也要看到风险：
   ...的风险依然存在。
   ```

---

### 六、结尾互动（30秒）

**目标**：引导点赞、评论、关注

**模板**：
```
以上就是今天的内容。

[总结一句话]

你觉得...？欢迎在评论区讨论。

如果这个视频对你有帮助，欢迎点赞关注，
我们下期见。
```

---

## 写作风格指南

### 语言风格

1. **口语化**：适合口播，避免书面语
   ```
   ❌ 鉴于上述分析，我们认为...
   ✅ 从这个数据来看，我认为...
   ```

2. **短句为主**：每句不超过20字
   ```
   ❌ 比亚迪在2024年第一季度实现了销量突破50万辆的成绩，这一数字同比增长了42%，创下了历史新高。
   ✅ 今年Q1，比亚迪卖了50万辆车。同比增长42%，创历史新高。
   ```

3. **数据具象化**：用对比让数据更有感知
   ```
   ❌ 比亚迪Q1销量50万辆
   ✅ 比亚迪Q1卖了50万辆车，相当于每天卖出5500辆，每分钟卖出4辆。
   ```

### 节奏控制

1. **开头要快**：前30秒必须有"钩子"
2. **中间要稳**：分析部分节奏均匀
3. **结尾要收**：不要拖泥带水

### 互动设计

在视频中加入互动点：
- "这个数据你没想到吧？"
- "你觉得谁会是下一个？"
- "如果换做是你，你会怎么选？"
